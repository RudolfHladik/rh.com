SWTOR Utilities

SWTOR Utility app lets you create your light side and dark side characters and save their different build setups. You can also browse every utility description of all classes. All these features with nice and clean material design. Since the app is still in testing stage if you want to try it you can download source code from my GitHub. Link is provided in contact section in the bottom of the page. Or you can email me and I will send you the .apk file. If you have any comment on function or design or feature advice feel free to email me as well.

LocSlides

LocSlides is web application completly developed with Dart and Polymer(0.5*). It is translated to JavaScript with Dart2JS library to be available on all major web browsers. It automaticaly generates content from two json files and picture folder. It was created to interactively show where the different groups of people come from and show theirs most popular sights and landmarks. In future I plan to make graphical interface to help deploy this app to any site.

*Polymer 1.0 is not yet available for Dart. As soon as it will be I'll try to migrate it.

rh-p-net

Simple reusable custom element, made with Polymer 1.0. It takes all <p> elements randomly resize them and randomly position them in the <rh-p-net> element so it requires to have appropriate width and height. It also tries all <p> elements to not overlap each other, but I'm still working on this feature. In future it should be able to choose limit points for random generated font-size values and positions via html attribute.
